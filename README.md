# Space Escape - Interactive Maze Game

A retro-style, space-themed 2D maze game built using vanilla JavaScript and HTML5 Canvas. Navigate your rocket through perilous asteroid fields to find a new home! 

*This project was created as part of my Multimedia class.*

## 🚀 Features
* **Grid-Based Movement:** Navigate a dynamic 10x10 grid using smooth, logic-driven canvas rendering.
* **Multiple Stages:** Progress through increasingly complex levels (includes Level 1 and Level 2 with planned trap mechanics).
* **Immersive Styling:** Features custom space-themed assets (asteroids, stars, rocket) wrapped in a neon-glowing CSS interface with the Orbitron font.
* **Collision Detection:** Strict boundary and wall-collision logic to prevent players from passing through asteroids.

## 🎮 How to Play
1. Clone this repository or download the project files.
2. Open `index.html` in any modern web browser.
3. Use the **Arrow Keys** (`Up`, `Down`, `Left`, `Right`) to move your rocket.
4. Reach the Finish portal to complete the stage and advance.

## 🛠️ Tech Stack
* **HTML5:** Canvas API for rendering the game grid and sprites.
* **CSS3:** Animations (`@keyframes glow`) and responsive styling.
* **JavaScript (Vanilla):** Game state management, 2D arrays for level generation, and event listeners for movement.
